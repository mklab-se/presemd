# Spatial Transition
Directional pan based on grid position

# Why Spatial?

- Slides feel like a 2D canvas
- Direction matches grid overview layout
- Navigate up, down, left, right

# Row Two, Slide One

This slide is on a new row in grid view.
Transitioning here pans vertically.

# Row Two, Slide Two

Moving between adjacent slides pans horizontally.

# Row Two, Slide Three

> Try pressing G to see the grid overview,
> then click different slides to see spatial panning.

## Summary

The spatial transition pans in the direction of the target slide's
position in the grid. Open grid view (G) to see the layout.
